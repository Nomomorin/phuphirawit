ยินดีต้อนรับสู่โปรเจ็คของฉัน

โปรเจ็คนี้เป็นเกี่ยวกับ ระบบดึงข้อมูลจากบัตรประชาชน

โปรเจ็คนี้จัดทำเพื่อลดเวลากรอกข้อมูลส่วนตัว เมื่อสถานที่ที่ต้องการข้อมูลส่วนตัวของผู้ใช้งานหรือลูกค้าอย่างเช่น โรงพยาบาล พัมนาด้วย python
(ข้อมูลส่วนตัวในหน้านี้ขออนุญาติปิดไว้นะครับ)

    ระบบที่มีหลักๆเลยคือ

  • api ดึงข้อมูลจากบัตรประชาชน
  • api หน้าเว็บไซต์แสดงข้อมูลบัตรประชาชน

    เนื่องจากมีการพัฒนาโปรเจ็คนี่ต่อ จึงเพิ่มระบบ

  • พัฒนาเป็นไฟล์ exe สามารถติดตั้งใช้งานได้เลย
  • ระบบเช็ตอัพเมื่อเปิดเครื่องทำให้โปรแกรมทำงาน ไม่ต้องไปเปิดใช้งานอีกรอบ
  • ส่งข้อมูลบัตรประชาชนข้ามแบบฟอร์ม
  • สามารถกำหนดปลายทาง URL ที่ต้องการส่งข้อมูลไปอีกฟอร์มได้
  • ระบบถอดการติดตั้ง

หน้าโปรแกรมเมื่อเปิดครั้งแรก จะให้ใส่ Url ที่ต้องการส่งข้อมูลไปอีกฟอร์ม

เมื่อเข้าเว็บที่กำหนดแล้วยังไม่ได้ใส่บัตรประชาชน จะแสดงว่าหาการ์ดไม่เจอ

เมื่อเข้าเว็บที่กำหนดแล้วได้ใส่บัตรประชาชน จะแสดงว่าใส่การ์ดแล้วพร้อมแสดงหน้าป๊อปอัพข้อมูล

ด้านล่างจะมีปุ่มยืนยันเมื่อต้องการส่งข้อมูลไปอีกฟอร์ม

Welcome to my project!

This project is about an ID card data extraction system.

t aims to reduce the time spent entering personal information when a location, such as a hospital, requires user or customer data. The project is developed using Python.
(Personal information on this page is intentionally omitted.)

    The core features of the system include:

  • API for extracting data from ID cards.
  • API for displaying ID card information on a website.

    As the project evolved, additional features were added:

  • Development of an executable file (exe) for easy installation and use.
  • System auto-start on machine boot, allowing the program to run without manual initiation.
  • Card data submission via a form.
  • Ability to specify a destination URL for sending the extracted data to another form.
  • Uninstallation feature.

When the program is opened for the first time, it will prompt the user to enter the URL to which they want to send the extracted data.

If a user accesses the specified website without entering an card, it will display a message indicating that the card is not found.

Once the user accesses the specified website and enters the card, the program will indicate that the card has been detected and display a popup with the extracted information.

At the bottom, there will be a confirmation button for sending the data to another form.
