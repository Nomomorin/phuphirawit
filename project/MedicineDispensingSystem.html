ยินดีต้อนรับสู่โปรเจ็คของฉัน

โปรเจ็คนี้เป็นเกี่ยวกับ ระบบจ่ายยาโรงเรียนเลิงนกทา

โปรเจ็คนี้จัดทำเพื่อใช้ในโรงเรียนลดเวลาในการลงชื่อจ่ายยา สามารถบันทึกข้อมูลฐานข้อมูลพัฒนาอยู่สามส่วนด้วยกันคือ frontend backend และ database อย่างแรก frontend เราพัฒนาด้วย framework Angular ส่วน backend พัฒนาด้วย framework NestJs ทั้งสอง framework ใช้ภาษา TypeScript

    ระบบที่มีหลักๆเลยคือ

  • ระบบข้อมูลนักเรียน( เพิ่ม, ลบ, แก้ไข, ดู)
  • ระบบข้อมูลยา( เพิ่ม, ลบ, แก้ไข, ดู)
  • ประวัติการจ่ายยา ที่สามารถนำข้อมูลออกมาในรูปแบบ pdf และ excel
  • ระบบจ่ายยา
