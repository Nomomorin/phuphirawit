Welcome to my project!

This project is about the Leang Nok Ta School Medication Dispensing System.

It is developed to streamline the process of medication distribution in schools, reducing the time spent on signing for medication. The system encompasses three main components: frontend, backend, and a database. The frontend is developed using the Angular framework, while the backend is built using the NestJs framework. Both frameworks utilize the TypeScript programming language.

    Key features of the system include:

  • Student Information System (Add, Delete, Edit, View)
  • Medication Information System (Add, Delete, Edit, View)
  • Medication Dispensing History: The system keeps track of the history of medication dispensing. Users can export this data in both PDF and Excel formats.
  • Medication Dispensing System

The usage of the website will feature various menus for users to navigate

Add Student Information Page: This page allows users to input and add information about students.

View All Student Information Page: Users can access this page to view information about all students.

View Medication Dispensing History Page: This page displays the history of medication dispensing, and users can export this data in both PDF and Excel formats.

Add Medication Information Page: Users can input and add information about different medications on this page.

View All Medication Information Page: This page allows users to view information about all medications.

Medication Dispensing Page: This page is dedicated to the actual process of dispensing medication.
